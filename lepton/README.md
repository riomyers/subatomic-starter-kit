# Lepton — Pro CLAUDE.md / AGENTS.md Blueprint Kit

Your CLAUDE.md is broken. You just don't know it yet.

60% of Claude Code support tickets are caused by bad context files — ghost context, conflicting rules, instruction bloat, stale references. Every wasted token in your CLAUDE.md costs you on every single message. A 500-line CLAUDE.md with 200 lines of dead weight means you're burning 40% of your context budget on noise.

Lepton fixes this.

## What's Included

### 8 Battle-Tested Templates
Production-grade CLAUDE.md templates for every project type. Not generic "write good code" garbage — specific, opinionated, proven instructions.

| Template | For | Key Focus |
|----------|-----|-----------|
| Solo Dev | Personal projects | Minimal, focused, fast |
| Team Project | Shared repos | Code ownership, PR standards, conventions |
| Frontend App | React/Vue/Svelte | Components, styling, performance, a11y |
| API Service | REST/GraphQL backends | Route design, validation, error handling, security |
| Monorepo | Turborepo/Nx/pnpm | Package boundaries, dependency management, builds |
| Data Pipeline | ETL/ML/analytics | Idempotency, data quality, schema contracts |
| DevOps | Infrastructure/CI/CD | IaC rules, deployment safety, monitoring |
| Open Source | Public repos | Contributor guidelines, API stability, releases |

### Cross-Platform AGENTS.md
Universal format that works on Claude Code, Gemini CLI, Cursor, and Codex. Write once, convert to any platform. Includes conversion guide for .cursorrules.

### Anti-Patterns Guide
The 20 most common CLAUDE.md mistakes — with before/after examples and specific fixes. Covers ghost context, context bloat, instruction quality, and structural issues.

### Validation Checklist
Pre-flight audit checklist with shell commands to programmatically check your CLAUDE.md for dead references, vague language, contradictions, and bloat.

### Migration Guide
Step-by-step process to audit, clean, and upgrade any existing CLAUDE.md. Includes scoring system, fix patterns, and maintenance schedule.

## Installation

```bash
npx @subatomic/cli install lepton --key YOUR_LICENSE_KEY
```

Templates install to `~/.claude/skills/lepton/` (Claude Code) or `~/.gemini/tools/lepton/` (Gemini CLI).

## Usage

1. Pick the template closest to your project type
2. Copy it to your project root as `CLAUDE.md` (or `AGENTS.md` for cross-platform)
3. Replace all `[BRACKETS]` with your project specifics
4. Delete sections you don't need
5. Run the validation checklist
6. Test in a fresh session
